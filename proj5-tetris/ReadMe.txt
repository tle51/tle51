- Composite design pattern was used was for this project.
- Files that are fully documented: shape.java the_score.java
- shape class is the Tetrominos class for this project

Case1:
The class XY acts as the composite class.
The class Shape uses XY class to store the coordinates
of each of its square blocks.

Case2:
The class the_score acts as the composite class.
The class inner_gui uses the_score class to update 
the score component  of the game.

How to use:
- Run the main class that is inside of the Simple_gui class.
