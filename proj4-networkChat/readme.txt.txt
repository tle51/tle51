Program Project 4: Network Chat

To chat to all users in the chat room, select the 'All' option from the drop down list to the left of the text box.
To send a private chat to a specific person, select the 'Private' option or use this format: /private <user name> <message>
To exit the chat window press the 'X' or type in /exit.
